## Objective

Install all AI SDK dependencies and create the environment variable template so that subsequent plans can import from `ai`, `@ai-sdk/google`, and `zod` without additional setup. Also create the shared `ParsedQuestion` Zod schema that both the parser (Plan 01-02) and the UI (Plan 01-03) will import.

Purpose: Unblock all downstream plans that import these packages or types.

**Step B — Create `lib/schemas.ts`:**

Create `lib/schemas.ts` with the following Zod schemas and exported TypeScript types:

```typescript
import { z } from "zod";

export const parsedOptionSchema = z.object({
  letter: z.string().min(1),       // "A", "B", "C", "D", "E"
  text: z.string(),                // option text content
});

export const parsedQuestionSchema = z.object({
  number: z.number().int().positive(),
  text: z.string(),                          // question body text
  options: z.array(parsedOptionSchema),      // answer choices
  imageBase64: z.string().nullable(),        // base64 PNG if question has image, null otherwise
});

export const solveRequestSchema = z.object({
  mode: z.enum(["no-bs", "verbose"]),
  questions: z.array(parsedQuestionSchema).min(1),
});

export type ParsedOption = z.infer<typeof parsedOptionSchema>;
export type ParsedQuestion = z.infer<typeof parsedQuestionSchema>;
export type SolveRequest = z.infer<typeof solveRequestSchema>;
```

These types serve as the contract between parser (Plan 01-02), UI (Plan 01-03), and API route (Phase 2).
