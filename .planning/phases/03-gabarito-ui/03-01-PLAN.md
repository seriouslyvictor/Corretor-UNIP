Create `components/gabarito-grid.tsx` as a `"use client"` component.

**Props interface:**
```typescript
interface GabaritoGridProps {
  parsedQuestions: ParsedQuestion[];
  solvedAnswers: SolvedAnswer[];
  isStreaming: boolean;
}
```

**Component structure:**

1. Header line: `<p>` with text `Gabarito ({solvedAnswers.length}/{parsedQuestions.length})` using `text-sm font-medium mb-3`.

2. Grid container: `<div className="grid grid-cols-5 gap-2">` iterating over `parsedQuestions` by index.

3. For each question index `i`:
   - Look up answer: `const sa = solvedAnswers.find(a => a.questionIndex === i)`
   - If `sa` exists (answered): render a filled cell
   - If `sa` is undefined and `isStreaming` is true: render a skeleton cell (per D-04)
   - If `sa` is undefined and `isStreaming` is false: render an empty/missed cell

4. **Filled cell markup:**
```tsx
<div className="flex flex-col items-center rounded-md bg-muted p-2">
  <span className="text-xs text-muted-foreground">{i + 1}</span>
  <span className={cn(
    "font-heading font-semibold",
    sa.confidence === "high" && "text-primary",
    sa.confidence === "medium" && "text-muted-foreground",
    sa.confidence === "low" && "text-amber-500",
  )}>
    {sa.answer}
  </span>
</div>
```
(per D-10, D-11: color on letter text only, cell background stays consistent)

5. **Skeleton cell markup (per D-04):**
```tsx
<div className="flex flex-col items-center rounded-md bg-muted p-2">
  <span className="text-xs text-muted-foreground">{i + 1}</span>
  <span className="h-5 w-4 animate-pulse rounded bg-muted-foreground/20" />
</div>
```

6. **Missed cell** (stream finished but no answer for this index): same as filled cell but show a dash `"-"` in `text-muted-foreground` instead of a letter.

**Exports:** Named export `export function GabaritoGrid(...)`.

**Imports needed:**
