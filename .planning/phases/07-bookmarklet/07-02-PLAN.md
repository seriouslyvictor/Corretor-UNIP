## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-07-06 | Spoofing | Bookmarklet anchor href | mitigate | BOOKMARKLET_HREF is a compile-time constant in the Server Component — it cannot be injected at runtime. No user input flows into the href. |
| T-07-07 | Tampering | javascript: href manipulation via XSS | mitigate | Page is a Server Component with no dynamic data; no user-controlled content is rendered into the href or page. XSS surface is zero. |
| T-07-08 | Information Disclosure | Session tokens in clipboard content | accept | Documented in on-page warning: "Only execute on ava.ead.unip.br." User is informed the pasted HTML may contain session-related content. Corretor does not persist the pasted HTML. |
| T-07-09 | Elevation of Privilege | User tricked into dragging fake bookmarklet from phishing page | accept | This is a social engineering risk against the user, not the application. The instructions page clearly attributes the bookmarklet to Corretor UNIP and specifies the UNIP domain. |
